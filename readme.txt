# Simulated Student Debate System

This project simulates a debate between AI agents with different personalities (dominant, passive, and reflective) to solve a machine learning problem.

---

## 📌 **Prerequisites**

Before running the code, ensure you have:

- **Python 3.8+** installed.
- **Git** installed.
- An **account on [Groq Console](https://console.groq.com/)** to get your API key.

---

## 🛠️ **Setup Instructions**

### 1. Clone the Repository

```bash
git clone https://github.com/tu-usuario/tu-repositorio.git
cd tu-repositorio
```

### 2. Create and Activate a Virtual Environment
```bash
python -m venv env
source env/bin/activate  # On Windows: `env\Scripts\activate`
```
### 3. Install Dependencies
```bash
pip install -r requirements.txt
```
### 4. Set Up Environment Variables
1. Create a .env file in the root of the project:
```bash
touch .env
```
2. Add your Groq API key to the .env file:
```bash
GROQ_API_KEY=your_api_key_here
```

3. Get your Groq API Key:

Go to Groq Console.
Sign in or create an account.
Navigate to the API Keys section.
Copy your API key and paste it into the .env file.

## ▶️ How to Run
Execute the main script:

```
python main.py

```
